# Дизайн-система @moysklad/uikit

Витрина компонентов библиотеки **@moysklad/uikit** `v28.1.0`.
Здесь можно «потрогать» компоненты живьём, посмотреть их варианты, токены
и готовые сборки экранов.

## Как устроена навигация

- **Токены** — фундамент системы: цвета, типографика, иконки.
- **Компоненты** — все ~75 компонентов, сгруппированы по назначению.
- **Демо-сборки** — примеры реальных экранов, собранных из компонентов.

## Подключение в своём проекте

```tsx
import '@moysklad/uikit/colorVariables.css'; // токены — один раз в точке входа

import { Button, ButtonVariants, ButtonSize } from '@moysklad/uikit';
import { Check20Icon } from '@moysklad/uikit/icon';
```
